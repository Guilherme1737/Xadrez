# Jogo de Xadrez - Java

Jogo de xadrez completo desenvolvido em Java puro, jogável no terminal.

##  Como Jogar

```bash
javac -encoding UTF-8 application/Program.java
java application.Program
```

Digite as coordenadas no formato **letra + número** (ex: `e2`, `e4`).
